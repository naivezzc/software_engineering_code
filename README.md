# Introduction
 The purpose of this repository is to preserve the source code from the software engineering course.
 We have developed a chess game with the following features:
 * Basic function
   * Initialize Board
   * Input function
   * Check if Input is vaild
   * Reverse function
   * count function
 * Extra function
   * Board visualize
   * Save and load game
